# 五大日常模組 SOP

這份給第一次上線的同仁用。先不用懂全部功能,每天只照這 5 個入口處理工作。

## 1. 商機追蹤

何時用:看到新標案、客戶詢問、轉介紹案源。

步驟:
1. 從「新標案通知」把有興趣的標案標起來。
2. 到「商機追蹤」按「匯入有興趣標案」。
3. 打開「今日追蹤」卡片,補截止日、勝率與最新紀錄。
4. 拖拉到正確階段:新機會、評估中、提案撰寫、已送件、得標、未得標、執行中、結案。

產出:每個案源都有客戶、金額、期限、階段、跟進紀錄。

## 2. 會計

何時用:收到款項、付廠商、建立報價、開發票。

步驟:
1. 先看「今日財務接續」,系統會提示先追應收、看報價或補交易。
2. 收款或付款按「新增交易」。
3. 報價階段按「建報價」。
4. 要請款按「開發票」。
5. 能綁工作包就填工作包 ID,之後可看專案毛利。

產出:交易、報價、發票、本月收入支出與淨利。

## 3. 會議速記

何時用:客戶會議、內部會議、電話錄音。

步驟:
1. 進「會議速記」按「上傳音檔」。
2. 可填工作包 ID,完成後才能一鍵推到交棒卡。
3. 等待語音轉文字與 AI 整理。
4. 完成後複製摘要貼到 LINE / Email,或推到工作包。

產出:會議標題、與會者、決議、待辦、關鍵數字、下次會議。

## 4. 場勘

何時用:活動現場、場地確認、搭建前檢查。

步驟:
1. 拍 1-5 張照片,可用 JPEG / PNG / WebP / HEIC。
2. 取得定位,補地址提示。
3. 可填工作包 ID。
4. 上傳後等 AI 產出場勘 brief。
5. 完成後可補 30 秒語音備註、複製 brief 或推到工作包。

產出:場地類型、空間估算、入口、洗手間、電源、停車、問題清單。

## 5. 使用教學

何時用:新人上手、忘記某個功能、要教同事。

步驟:
1. 先選你的角色。
2. 看角色進度清單。
3. 搜尋你要做的事,例如「會議」、「報價」、「場勘」、「商機」。
4. 照這份 SOP 完成第一輪真實任務。

建議第一天完成:
1. 建一筆商機。
2. 建一筆報價。
3. 上傳一段短會議音檔。
4. 上傳一張場勘照片。
5. 複製一次交棒摘要。
