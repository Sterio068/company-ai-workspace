# 📇 同仁帳號管理 · 老闆 SOP

> 給管理員 · 加 / 改 / 刪同仁 全流程 · 含 PDPA 注意事項

---

## 一、加新同仁

1. 中控(⌘M)→ 點上方「**同仁管理**」連結 · 或從 sidebar 進
2. 右上角 **「+ 建新同仁」**
3. 填基本資料:
   - **電子郵件**(同仁登入用 · 建後不可改)
   - **顯示名稱**(王小明)
   - **初始密碼**:
     - 自己打,或按 **🎲 隨機產** 產 16 字強密碼
4. 選 **頭銜**(會計 / 企劃 / 設計…)+ **角色**(一般同仁 / 管理員)
5. 套 **預設模板**(右上下拉)· 7 個職務模板自動勾權限
6. 微調權限(可選):28 個細部權限
7. **建立** → 跳出框框顯示 email + 密碼,**📋 複製** → 貼給同仁

---

## 二、改頭銜 / 權限

1. 同仁管理 表格 → 找該同仁 → 「**編輯**」
2. 改 顯示名稱 / 頭銜 / 角色 / 權限
3. **儲存**

> ⚠ email 建後**不能改** · 要改請刪重建

---

## 三、🔑 重設密碼(同仁忘記了)

> 同仁向老闆求救「我忘記密碼」時用

1. 同仁管理 → 找該同仁 → 「**編輯**」
2. modal 中段「🔑 密碼:同仁忘記了 / 老闆要硬覆蓋」→ 點 **🔑 重設密碼**
3. 自動產一組新密碼(可手改)
4. **確認重設**
5. 跳出框框顯示新密碼 → **📋 複製** → 貼給同仁
6. ⚠ 舊密碼立即失效 · 同仁需用新密碼重新登入

---

## 四、💤 停用(離職 / 暫離)

> 軟刪除 · 帳號不能登入,但所有歷史資料保留(對話 / 專案 / 場勘)

1. 同仁管理 → 該同仁列右側 → 「**停用**」
2. 確認框 → 「停用」

> 反悔 → 點「**復用**」即恢復登入

---

## 五、🗑 永久刪除(PDPA · 個資請求)

> 同仁要求依個資法刪除個人資料時用 · **無法復原**

#### 必要前提

1. 同仁帳號**已先停用**(紅色按鈕「停用」)
2. 二段式確認:警告 → 打字驗證 email

#### 操作

1. 同仁管理 → 找該同仁(灰色 = 已停用)→ 「**🗑 刪除**」
2. 第一步 · 警告框 → 「我了解 · 下一步」
3. 第二步 · 完整輸入 email 字串 → 「永久刪除」
4. 完成 toast 顯示「連帶清 sessions=N / feedback=N」

#### 會刪什麼

- ✅ users 同仁帳號(含密碼 hash)
- ✅ sessions / refreshTokens 登入 cookie
- ✅ feedback 👍👎 紀錄

#### 會保留什麼(法律抗辯需要)

- 對話紀錄(conversations)
- 操作 audit log

> 真要清對話 → 走 PDPA 完整流程(管理員 · 看 docs/05-SECURITY.md「人員異動」章節)

## 六、權限模板對照

| 模板 | 適用 | 主要權限 |
|---|---|---|
| 老闆 / Champion | 全權 | 全部 28 項 |
| 企劃 / 專案經理 | PM | 投標 / CRM 全 / 專案 全 / 設計 / 公關 / 會議 / 知識 |
| 設計師 | 創意 | 專案(自己)/ 設計 / 素材 / 會議 / 場勘 / 知識 |
| 公關 / 媒體窗口 | 對外 | 新聞稿 / 社群 / 媒體 CRM / 會議 |
| 會計 / 財務 | 後勤 | 會計 全 / 會議 / 知識 |
| 業務 / 顧問 | BD | 投標 / CRM(自己)/ 專案 / 公關 / 會議 |
| 實習生 / 新人 | 最小權 | 專案(自己)/ 設計 / 會議 / 知識 |

> 進階:選好模板後**還可以再勾 / 取消個別權限**

## 七、安全提醒

### 🔴 不能做

- 不能停用 / 改自己的 email(防 admin lockout)
- 不能改自己的密碼從這個介面(走右上角頭像 → 個人設定)
- 不能直接 hard delete 沒先停用的帳號(二段防誤操作)

### 🟢 建議做

- 新同仁建好密碼**當面 / LINE 給** · 不要 email(避免被攔)
- 同仁離職:**先停用 24 小時**確認沒事 → 再永久刪除
- 每月看一次列表 · 該停的停 · 該升降權限的調

## 八、常見問題

### Q1 · 我手滑停用了現職同仁怎麼辦

點同仁列右側「**復用**」即可恢復 · 完全無損失。

### Q2 · 永久刪除後對話紀錄還在?

是 · 為法律抗辯 / 商業機密保留 · 請依公司資料保留政策 + PDPA 流程處理。

### Q3 · 同仁可以自己改密碼嗎?

可以 · 登入後 → 右上角頭像 → 個人設定 → 改密碼

### Q4 · 可以加多個 admin 嗎?

可以 · 編輯同仁 → 角色選「🔴 管理員」即可。建議至少 **2 位 admin** 防 lockout(但同時也是資安風險,要謹慎)。
