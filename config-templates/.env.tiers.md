# Environment Variable Tiers

日期:2026-04-28
Source of truth:`backend/accounting/services/secret_registry.py`

## T0_REQUIRED

沒有這些值,正式環境應 fail closed 或無法安全交付。

| Key | Reader | Writer |
|---|---|---|
| `OPENAI_API_KEY` | LibreChat + accounting tools | installer/keychain 或 admin UI(accounting fallback) |
| `JWT_REFRESH_SECRET` | LibreChat + accounting auth | installer/keychain only |
| `ECC_INTERNAL_TOKEN` | accounting internal endpoints + scripts | installer/keychain only |
| `MEILI_MASTER_KEY` | Meilisearch + LibreChat | installer/keychain only |
| `CREDS_KEY` | accounting oauth token service | installer/keychain only |

## T1_RECOMMENDED

核心體驗建議設定,但系統可用 fallback 或保持 local-ready。

| Key | Reader | Writer |
|---|---|---|
| `ANTHROPIC_API_KEY` | LibreChat | installer/keychain |
| `NOTEBOOKLM_ACCESS_TOKEN` | accounting NotebookLM bridge | admin UI(寫入前驗證)或 keychain |
| `NOTEBOOKLM_PROJECT_NUMBER` | accounting NotebookLM bridge | admin UI 或 `.env` |

## T2_OPTIONAL

加值能力或營運設定。

| Key | Reader | Writer |
|---|---|---|
| `FAL_API_KEY` | accounting design route | admin UI |
| `IMAGE_PROVIDER` | accounting design route | admin UI |
| `EMAIL_USERNAME` | accounting mailer | `.env` / keychain |
| `EMAIL_PASSWORD` | accounting mailer | `.env` / keychain |

## Rules

- 前端管理 UI 只能寫 `frontend_writable=true` 的項目。
- 認證、internal token、加密 key 一律走 installer / macOS Keychain。
- NotebookLM token 寫入前會先打官方 API 做 lightweight 驗證;失敗回 `412` 並保留原設定。
- 更新 registry 時同步更新本文件與安裝流程提示。
