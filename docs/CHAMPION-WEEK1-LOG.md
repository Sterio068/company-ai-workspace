# Champion 第一週日誌範本 · 承富 AI 系統

> **使用對象:** 承富指定的 1-2 位 Champion
> **目的:** Day 0 起 · 連續 7 天每日 5-10 分鐘記錄 · 第 7 天交給 Sterio + 老闆
> **價值:** 這份日誌是「上線是否成功」的唯一第一手資料(不是後台數字 · 是現場觀察)

---

## -1. Champion 選拔規則(Day -7 前老闆決策)

### 選拔順序(三層 · 上層不行再下層)

```
🟢 Layer 1 · 自願優先
    公開徵求:「誰願意每天花 30 分鐘當 Champion · 算正式工時 + 列績效」
    有人舉手 → Champion + 副 Champion 定案 · 直接 Day 0

🟡 Layer 2 · 老闆指派(最多 1 位 · 不要逼)
    若 Layer 1 沒人或只有 1 位 · 老闆指派
    但指派對象若口頭不爽 · 立刻放棄 · 走 Layer 3
    ⚠ 絕不硬派 · 日誌會變形式作業 · 失敗率反而升高

🔴 Layer 3 · 縮 Pilot 模式(3-5 人先上)
    老闆選用得最多 / 最主動的 3-5 人先上線
    其他 5-7 人第 2 週才進入
    Pilot 成員「只記錄不扛客服」· 老闆直接指派任務
    這 3-5 人就是天然 Champion(不掛名)
```

### Layer 1 · 自願者篩選條件(至少 3 項)

| 條件 | 為什麼 |
|---|---|
| ☐ 不是老闆 · 不是 Sterio | 老闆太遠 · Sterio 是外包 |
| ☐ 在公司待 > 6 個月 | 對流程夠熟 · 能判斷「省時」真假 |
| ☐ 對新工具有熱忱(過去主動學過 1 個) | 否則裝不出興趣 |
| ☐ 大家願意問他事情的人 | 服眾 · 不然沒人聽 |
| ☐ **不是** 正在忙 Q2 大案的 PM | 沒時間 · 撐不過 7 天 |
| ☐ 聲量適中 · 不是最安靜也不是最喧鬧 | 過靜無說服力 · 過喧引反感 |

### Layer 3 · 縮 Pilot 模式啟動條件 + 規則

**啟動條件:** Day -7 時 Layer 1 無自願者 + Layer 2 指派對方婉拒
**時程影響:** 不延後上線 · 但規模縮至 3-5 人

**Pilot 3-5 人選法(老闆主導):**
1. 近 1 個月**對效率工具主動發問最多**的
2. **已私下用過** ChatGPT / Claude 個人版的
3. 本週**案子不特別忙**的(不是 dead week)

**Pilot 規則:**
| 項目 | Pilot 模式做法 |
|---|---|
| 上線人數 | 3-5 人 · 不是 10 人 |
| Day 0 時程 | 上午照常(9:00-12:00) · 下午縮成 13:30-15:30 |
| first-win 硬條件 | 維持 7/10(於 5 人 = 4 人 · 於 3 人 = 3 人全) |
| Champion 角色 | 老闆自己當「隨時在場的指派人」· 不寫日誌 · 改寫 5 案抽樣 |
| 客服責任 | **無 Champion** · 大家直接問 Sterio(Sterio Day 0 全天在) |
| 第 2 週擴展 | 若 Pilot 4/5 有具體省時案 · 第 2 週擴至全員 |

**Pilot 模式的日誌改法(不適用本檔全部 · 改用 §9 精簡版):**

```markdown
# Pilot Day +__ · 2026-__-__
今日使用者:__/5
今日成功案例:1._____ 2._____
卡點:1._____ 2._____
本週是否擴展?___(由老闆 Day +5 決定)
```

1. 老陳完全沒登入 · LINE 問了沒回 · 明天午餐主動找
2. 小華貼了招標 PDF 但**只貼半份** · 教她「全選 → 複製」

## 4. Champion 自己這週的 self-care

寫日誌 7 天會累 · Champion 不是工具人 · 自己也要照顧:

| 提醒 | 做法 |
|---|---|
| 不要當客服 24/7 | Slack 設定 OOO 18:00-09:00 · 同仁私訊請等隔天 |
| 不要替同仁操作 | 教他怎麼操作 · 不要直接幫他點 · 否則他永遠學不會 |
| 不要扛工程責任 | 同仁說「這個爛」· 你回「我會跟 Sterio 反映」· 不要承諾改 |
| 自己要有 first-win | Champion 自己每週要產 1 個成功案例 · 否則沒說服力 |
| 找 Sterio 喝咖啡 | Day +7 跟 Sterio 真的見面 · 不只是 Slack |

## 5. Champion 的「禁區」

不要做的事:

- 🚫 不要透露其他同仁的對話內容(即使對方同意)
- 🚫 不要替老闆做「誰用得多誰用得少」的排名(數字交給 admin dashboard)
- 🚫 不要修 prompt(那是 Sterio 的事 · 你只回報問題)
- 🚫 不要偷加自己的個人額度(超預算找 Sterio · 不是自己 hack)
- 🚫 不要在 Day +1 就罵同仁不會用(第一週是學習期)
