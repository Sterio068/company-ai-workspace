# 04 · 業務一件事(5 分鐘)

> 你最累:每天盯政府採購網找新標案、跟客戶 LINE 報告、寫比價信給廠商。
> 這份讓你**每天省 1 小時**。

---

## 1 招速記

**每天早上 9 點,開承富 AI 按 ⌘T,5 分鐘看完全台灣有可投的新標案。**

---

## 每天要做的 3 件事

### 早上 9:00 · 看新標案(5 分鐘)

按 **⌘T** 進標案監測頁。系統每天凌晨自動從政府採購網抓新標案 · 符合承富關鍵字的才列(例如:宣導、公關、活動、設計、行銷)。

對每一條標案你只做 3 件事:
- **✨ 有興趣** → 按「有興趣」· 匯入商機 Pipeline
- **🗑 跳過** → 按「跳過」· 之後不再顯示
- **不確定** → 留著,明天再看

**之前:手動登入政府採購網 + 篩 20 分鐘** → **現在:5 分鐘滑完列表**

### 早上 9:10 · 看商機 Pipeline(2 分鐘)

按 **⌘I** 進商機頁。你的標案以漏斗視覺化:

```
新機會 → 評估中 → 寫建議書 → 送件 → 得/落標 → 執行 → 結案
```

每個卡片是一個案子 · 拖拉到對應階段。老闆一看就知道漏斗哪一段卡。

### 下午 · 跟客戶 LINE 回報(10 分鐘)

客戶 LINE 問進度,你以前要邊想邊打、怕漏。

**現在:** 按 ⌘Enter 問首頁主管家:
```
客戶 XX 問我們案子進度。我們目前在提案撰寫階段 · 預計 X 月 X 日送件。
幫我寫 LINE 回覆 · 不要太官腔,但要專業。
```

助手給你 3 段:
1. 打招呼 + 感謝關心
2. 目前進度(1 句)+ 下一步(1 句)
3. 結尾 + 保持聯絡

複製貼上 LINE · 送出。

---

## 跟廠商比價的省時招

### 活動要訂 5 家廠商比價(音響 / 燈光 / 餐飲 / 直播 / 布置)

按 **⌘2**(活動執行助手),說:

```
2026 X 月 XX 記者會 · 台北 · 200 人 · 3 小時
幫我寫給 5 類廠商的標準比價信:音響 / 燈光 / 餐飲 / 直播 / 現場布置。
每類給我可 copy 的 email(含主旨、規格、回覆期限)。
```

助手給你 5 封 email · 你:
1. 填入你心目中那 5 家廠商的 email
2. 人工看一次規格有沒有漏
3. 分別送

**之前:5 家各打一封信,1 小時** → **現在:10 分鐘**

---

## 每週要做的 1 件事

### 週五下午 · 週報

按 **⌘5**(營運助手),貼:
```
這週我做了:
- 判 3 案(環保、長照、文化)、2 Go 1 NoGo
- 送出 2 份建議書
- 客戶 A 結案會議
- 廠商比價信 8 封

幫我寫週報給老闆 · 300 字內 · 看得出產出量。
```

助手給你 300 字週報 · 你 copy 貼 Email 給老闆。

## 小 tips

### LINE 客戶訊息複雜不知怎麼回

整段複製貼進對話框 + 問:
```
客戶傳這段給我,幫我判讀真意 + 寫 3 個不同語氣的回覆(強硬 / 中性 / 討好)。
```

### 商機 Pipeline 拖拉很卡

拖卡片到新階段時 · 系統會打後端 API · 網路慢會看起來卡。等 1 秒就好。

### 標案關鍵字想加

跟 Champion / 外包工程師講一聲(例如「想加『永續』『ESG』」) · 他改 cron 腳本,隔天生效。
