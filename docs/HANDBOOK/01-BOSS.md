# 01 · 老闆快速上手(5 分鐘)

> 不用懂技術。這份讓你看得懂全公司用得怎樣、每月花多少。

---

## 你每天只要看一個畫面

**打開 <http://localhost/> · 登入後按 ⌘M**(管理面板)

會看到:

### 💰 本月財務
- 收入 / 支出 / 淨利 / 進行中專案
- 從會計模組即時撈 · 不用等會計小姐做月報

### 👍 品質指標
- 整體滿意度 %(同仁按 👍/👎 累積的)
- 各助手品質排行 · 差的會立刻知道
- 本月對話數 · 誰用的多誰少

### 🤖 AI 用量
- 近 30 天各模型(Haiku / Sonnet / Opus)用了多少 tokens
- 每次呼叫多少錢
- **預算 NT$ 12,000 / 月 · 首頁進度條到 80% 會變橘、100% 變紅(你自己留意)**
- ⚠️ 目前是**儀表提醒**,還沒自動 email / 自動暫停 — 這些在 v1.1 上(Champion 會協助每週 check)

---

## 你最常問的 3 個問題

### Q1. 「這個月花多少?」

⌘M → 看「本月財務」淨利。或按左側「💰 會計」,看本月 P&L。

### Q2. 「我們哪個助手同仁最常用 / 最準?」

⌘M → 看「各助手品質排行」。滿意度 < 60% 的,找 Champion 一起看是哪裡不準。

### Q3. 「新標案有什麼可以投?」

按 ⌘T(標案監測) · 系統每天早上自動從政府採購網抓新標案 · 符合承富關鍵字的才列。

---

## 同仁抱怨「AI 不準」時,你看這裡

⌘M 下方「各助手品質排行」。按 👎 率排序 · 前 3 名就是同仁的痛點:

- **👎 率 > 40%** · 找 Champion 檢查 prompt 設定
- **👎 率 20-40%** · 可接受 · 月底自動會產優化建議
- **👎 率 < 20%** · 維持就好

---

## 你不用碰的事

- **建新助手 / 改 prompt** → Champion 或外包工程師
- **備份 / 系統維運** → 每日自動 · 出事會 email 通知你
- **新同仁帳號建立** → 找 Champion

---

## 只要記住 3 個快捷鍵

| 按鍵 | 做什麼 |
|---|---|
| **⌘M** | 管理面板(預算 / 品質 / 用量) |
| **⌘T** | 新標案 |
| **⌘K** | 搜尋任何東西 |

---

## 若系統完全打不開

- 打電話給:AI Champion 或外包工程師
- 簡訊:「承富 AI 打不開」· 他們收到 15 分鐘內會處理
- 這段期間同仁可用自己的 ChatGPT / Claude web · 照常工作
