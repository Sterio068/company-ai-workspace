# 02 · PM 三件事(10 分鐘)

> 承富最累的角色:從看到招標到送件、從開案到結案,你都要扛。
> 這份教你 3 招 · 每招省 1-2 小時。

---

## 3 招速記

1. **看到招標** → ⌘1 投標助手 → 10 分鐘判 Go/No-Go
2. **提案撰寫** → ⌘1 投標助手 → 5 章初稿 2.5 小時
3. **結案報告** → ⌘5 營運助手 → 框架 + 填內容 1 小時

完整走一遍請看 [docs/CASES/01-海廢案端到端.md](../CASES/01-海廢案端到端.md)。

## 招 1 · 判標 10 分鐘

### 老闆傳你一份招標 PDF,以前你要花半天讀

**現在:**
1. 按 **⌘1**
2. 把招標內容複製貼到對話框(v1.1 會開 PDF 上傳)
3. 貼這段 + 按 Enter:

```
幫我 Go/No-Go 判斷。承富是 10 人公關行銷(政府案成案率 60%)。
我有 [X] 週可準備。
```

4. 助手會回:8 維度評分 + Go/No-Go 建議 + 第 1 週 3 件事

### 卡住時用這 3 句救

- 「不要解釋,直接評分」
- 「承富過往做過 XX 案,重評」
- 「只列 3 個最關鍵風險」

## 招 2 · 建議書 5 章 2.5 小時

### 得標決定 Go 後,第一件事是寫 5 章建議書

繼續在同一個對話(助手會記得上一步的 Go/No-Go):

```
好 · 投。幫我寫建議書第 1 章「需求理解與詮釋」約 800 字。
公文體 · 最後一段帶到「承富是最適合執行者」。
```

依序問第 2-5 章:
- 第 2 章:**策略方向**(呼應第 1 章需求)
- 第 3 章:**執行方法**(含甘特圖描述)
- 第 4 章:**團隊介紹**(你自己寫最快)
- 第 5 章:**預算分配**(跟助手要「預算 XX 萬拆 4 類」)

**每章寫完做 3 件事:**
1. 讀一次,改掉不符承富的用詞
2. 存成 docx(複製到 Word 貼上)
3. 放進專案資料夾

## 招 3 · 結案報告 1 小時

### 案子結案時,你最想的是快點寫完回家

按 **⌘5**(營運助手),貼:

```
[案名] 結案了,幫我產 4 章結案報告框架:
成果 / 數據 / 見解 / 附錄。
KPI:媒體曝光 X 萬、社群觸及 Y 萬、現場 Z 人。
```

助手給你框架 + 關鍵填空 · 你填 1 小時搞定。

## 4 個日常小 tips

### 1. 找過往案例

按 **⌘K** → 打關鍵字(例如「淨灘」、「記者會」)· 找得到就直接 re-use 段落。

### 2. 跟設計師溝通

按 **⌘3** → 把客戶要求貼給助手 → 產結構化 Brief → 轉貼給設計師。

### 3. 廠商比價

按 **⌘2** → 說「請 5 家音響廠商報 XX 活動」· 產可 copy 的信。

### 4. 客戶回覆 LINE 訊息

LINE 收到客戶訊息不知怎麼回?按 **⌘Enter** 問首頁「✨ 主管家」:
```
客戶 LINE:「...」 幫我寫回覆 · 專業但不冷淡。
```

## 同仁常問我

### 「PDF 要打字貼進去太麻煩」
v1.1 會開附件上傳。現在暫時:用 Preview.app 打開 PDF → ⌘A 全選 → ⌘C → 貼進對話框。

### 「AI 回答跟我們公司風格不一樣」
第一次問加一句:「承富是 10 人公關行銷,政府標案為主,公文體」。它會記這次對話。

### 「我要交接對話給同事」
目前靠複製對話內文。v1.1 會做「專案分享」。
