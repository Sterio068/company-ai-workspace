# 承富 AI · 10 分鐘上手

> 不需要任何電腦知識。只要會用 LINE、會複製貼上,就會用。
> 讀完這頁,你會做 3 件平常花 1-2 小時的事,3 分鐘搞定。

---

## 你會得到什麼

- **設計 Brief 3 分鐘產出**(平常 90 分鐘)
- **招標 PDF 10 分鐘判值不值得投**(平常半天)
- **廠商比價信一次寫好 5 家**(平常逐家打)

---

## 第 0 步 · 打開承富 AI(30 秒)

在瀏覽器打 <http://localhost/> · 按登入。

第一次登入會跳 3 步任務導覽 —— **認真做完它**,5 分鐘搞定,之後就會用。

若畫面卡住 · 按 ⌘⇧R 硬刷新一次。若完全打不開 · 找管理員。

---

## 第 1 步 · 用設計助手(3 分鐘)

**場景:** 客戶要中秋節社群活動,你要跟設計師開 brief。

1. 首頁點紫色「🎨 設計協作」卡(或按 ⌘3)
2. 在輸入框貼這段:

```
幫我想 3 個中秋節社群活動主視覺方向。
客戶是 XX 食品,品牌色是橘黃色,預算 5 萬,3 天內要。
我要 FB 1:1、IG Story 9:16、LINE 3:4 三個尺寸的素材清單。
```

3. 按 Enter 送出
4. 3 秒後會看到助手回你:3 組方向名稱 + 色調 + 各尺寸素材清單

**你就完成了設計 Brief 初稿。** 把這段直接貼給設計師,他會愛你。

---

## 第 2 步 · 用投標助手(10 分鐘)

**場景:** 老闆丟你一份招標 PDF,問值不值得投。

1. 首頁點紅色「🎯 投標」卡(或按 ⌘1)
2. 把招標 PDF 整段複製貼到輸入框(目前先複製貼上 · v1.1 會開檔案上傳)
3. 在末尾加一句:

```
幫我 Go/No-Go 判斷,我們是承富創意(10 人公關行銷),
過往政府案成案率 60%。我有 8 週可準備。
```

4. 按 Enter

**助手會回你:**
- 8 維度評分表(難度 / 競爭 / 預算 / 時程 / 匹配度 / 風險 / 承富優勢 / 總分)
- 明確 Go / No-Go / 有條件 Go 建議
- 如果 Go,第一週要做的 3 件事

**如果 AI 偏題了** · 貼這句把它拉回來:

> 不要解釋概念。直接告訴我 Go 還是 No-Go,和 3 個理由。

---

## 第 3 步 · 用活動助手(5 分鐘)

**場景:** 要發比價信給 5 家音響廠商。

1. 首頁點橘色「🎪 活動執行」卡(或按 ⌘2)
2. 貼這段:

```
幫我寫一封給音響廠商的報價詢問信。
活動:2026 X 月 XX 記者會 · 200 人 · 3 小時 · 台北市某飯店。
需求:主持人麥 2 支 + 貴賓麥 4 支 + 音箱覆蓋 200 人座位。
請給我可以直接 copy 的 email 格式(含主旨、稱謂、結尾)。
```

3. 送出後,助手會給你:**主旨 + 問候 + 規格 + 時程 + 回覆期限**

4. 把這封 email **自己看過一次確認沒漏**,再分別發給 5 家廠商。
   (v1.1 會加「一鍵群發+自動收集比價」)

| **⌘K** | 全域搜尋 · 找助手、工作區、專案、範本(對話搜尋 v1.1 開放) |

**整個系統搞不清楚** · 直接問「主管家」(首頁最上面那個 ✨ 按鈕):

> 我是 XXX,要做 YYY,我不知道該用哪個助手。

它會幫你判斷該找誰。

---

## AI 回答不好怎麼辦?

不要直接重問 · 改用這 3 招:

1. **加具體範例** · 「像 OO 那種風格」、「參考我們 2024 年 XX 案」
2. **加限制條件** · 「不要用列點,給我 3 段文字」
3. **給 👎** · 在回答下方點 👎 按鈕 · 承富 AI 月底會自動學
