> 適用 Agent:#02 活動規劃師 · #00 主管家(現場 PM 求救時)

## 通用 5 分鐘決策框架

任何危機:
1. **0-30 秒**:STOP 不要慌 · 深呼吸
2. **30s-1min**:確認 3 件事
   - 安全嗎?(人身 / 法律)
   - 能不能 5 分鐘內解?(自處 vs 升級)
   - 客戶 / 媒體在現場嗎?(公關層級)
3. **1-3min**:決策樹
   - 安全 ✓ + 自處 ✓ + 媒體 ✗ → **靜悄悄修** + 事後簡報
   - 安全 ✓ + 自處 ✗ + 媒體 ✓ → **公開「我們處理中」+ 立即啟動 backup**
   - 安全 ✗ → **撤離 + 報警 + 通報**
4. **3-5min**:執行 + 通報 PM 主管
5. **5min+**:每 15 分鐘給上級簡報進度

## 詳細 SOP

### 場景 1 · 主持人遲到 / 缺席
**5 分鐘內**:
1. 通知客戶 → 詢問有沒有副手
2. 若無 · 看現場誰能撐(資深 PM / 客戶高層 / 工作人員)
3. 改 timeline:把開場往後 10 分鐘
4. 公開廣播:「貴賓還在路上 · 我們先進場暖場」
5. 啟動暖場活動(影片循環 / 音樂 / 自由交流)

**避免**:不要提主持人名字 · 不要說「等他」· 不要顯示焦慮

### 場景 2 · 設備壞
**5 分鐘內**:
1. 投影壞 → 立刻換筆電 + HDMI 線備援(每場必有)
2. 麥克風壞 → 切備援麥(每場必有 1 組備)
3. 燈壞 → 開全場日光燈 + 緊急加燈
4. 全部備援都壞 → 改現場麥(無線 mic 變大聲版本)+ 紙本資料

**事前**:每場活動「3 備援」(投影 / 音響 / 電源)清單(skill 07 場勘 checklist)

### 場景 3 · 現場觀眾 < 預期 70%
**5 分鐘內**:
1. **不要在最前面講「人怎麼這麼少」**
2. 把椅子往前縮 · 中間留空(視覺密度)
3. 開場時說:「今天精選了 X 位最相關的貴賓」(reframe)
4. 縮短議程(原 60 分 → 45 分)· 讓互動 / Q&A 變主軸
5. 拍照角度從低處往上 · 避免空椅子入鏡

**事後 1h**:寫客戶紀錄(skill 11)分析原因 · 下次校正

### 場景 4 · 現場衝突 / 抗議者
**0-1 分鐘**:
1. **不要靠近 / 不要對質**
2. 立刻通知警察 + 場地保全
3. PM 上前用平靜語氣:「您好 · 這是私人活動 · 請您先到外面 · 我們可以聽您意見」
4. 同時通知:客戶 PR + 公司公關 + 上級主管
5. 引導其他賓客遠離(不要圍觀)

**避免**:不要拍照 / 不要錄影對方(法律灰色)
**1 小時內**:寫情況報告(時間 / 地點 / 人 / 經過 / 結果)交給客戶 PR

### 場景 5 · 餐飲遲到 / 餐損
**5 分鐘內**:
1. 確認還要多久 → 給時間
2. 啟動填補:咖啡 / 點心區先開
3. 廣播:「主餐稍晚 · 先到旁邊用點心」(reframe)
4. 餐損(餐量不夠 / 食物事件):
   - 食安疑慮 → 立刻停 · 通知餐廳 · 改備案
   - 量不夠 → 緊急加單(便利商店 / 手搖)填補

### 場景 6 · VIP 臨時取消
**1 小時內**:
1. 不要立刻在公開場合宣布
2. 通知客戶 → 確認替代方案
3. 議程上 VIP 段落改:
   - 換人(同份量級 · 客戶協助)
   - 改播 VIP 預錄影片
   - 取消該段 + 縮短後續
4. 若 VIP 是核心吸引點 → 退費 / 補償方案要先準備

### 場景 7 · 天災 / 大停電
**0-5 分鐘**:
1. 安全第一 · 引導觀眾到避難處
2. 啟動緊急照明(每場活動必備)
3. 保護現場貴重物 + 客戶資料
4. 啟動緊急應變網:警消 / 場地經理 / 客戶
5. 若延期 → 立刻發 LINE / Email 通知所有報名人

## 紅旗 · 不可接的危機

絕對升級給法務:
- 涉及人身傷亡
- 媒體現場拍 + 開始發布
- 客戶 / 競品在抗議現場
- 涉及政府高官 / 立委
- 食安重大事件

通報順序:現場 PM → 公司 CEO → 法務 → 客戶 PR

---

## 危機後 24h 必做

1. **書面紀錄**:時間軸 + 對話 + 決策 + 後果
2. **影像存證**:現場照片 / 監視器(如有)
3. **客戶簡報**:當天簡訊 + 隔日完整 PDF
4. **內部復盤**:會議 30 分鐘 · 寫進 skill 12 結案報告
