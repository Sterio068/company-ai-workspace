# Skill 15 · 結案 KPI 報表標準

> 案子做完寄客戶的「成果報告」標準格式
> 適用 Agent:#09 結案營運 · #04 公關寫手(若客戶要對外報導)

---

## 報表類型選對

|案類|要的 KPI|錯選會被打槍|
|---|---|---|
|**政府標案**|履約事項 100%(QA 表)+ 量化指標 + 額外 + 經費執行率|送主觀感受 / 設計美學 → 政府不收|
|**企業活動**|觸及人數 + 媒體曝光 + 互動率 + ROI(估算)|只送出席名單 / 沒對標 KPI|
|**公關專案**|媒體 placement 數 + AVE + sentiment + 競品比較|送內部事件記錄 / 客戶不在乎|
|**設計專案**|交付物清單 + 修改輪數 + 客戶簽收 + 後續授權|沒簽收 → 帳收不回|
|**長期合約**|月度進度 + 累積成果 + 下季規劃|沒下一期建議 → 客戶不續約|

## 標準 4 章結構

### 章 1 · 摘要(1 頁 · 給高層看)
公式:
```
【案名】[完整全銜]
【期間】YYYY-MM-DD ~ YYYY-MM-DD
【執行率】XX%(履約 + 額外)

# 一句話成果
[本案達成 / 超越 / 未達 X 項目標 · 主要 KPI 是 Y]

# 三大亮點
1. [量化 + 動詞 + 對象] → 例「媒體曝光較預期 +35% · 觸及 4.2 萬人」
2. [量化 + 動詞 + 對象]
3. [量化 + 動詞 + 對象]

# 三個學到的事
1. [客戶可後續用的 lesson]
2. [量化教訓]
3. [意外發現]
```

**規則**:
- 只能 1 頁 · 高層 30 秒能讀完
- 數字一律具體(不可「很多」「不錯」「應該」)
- 三大亮點 vs 三個學到 平衡 · 不全是好

### 章 2 · 數據(2-4 頁 · 圖表為主)

**3 個必含 chart**:

1. **目標 vs 實際對照表**(由 #07 財務試算 產 Excel)
```
| KPI | 目標 | 實際 | 達成率 | 備註 |
|---|---|---|---|---|
| 觸及人數 | 3 萬 | 4.2 萬 | 140% | 超越 |
| 媒體曝光 | 8 篇 | 12 篇 | 150% | 超越 |
| 預算執行率 | 100% | 97.3% | — | 略剩 |
| 客戶滿意度 | 4.0 | 4.5 | 113% | 5 分制 |
```

2. **時間軸進度**:Gantt 風 · 顯示「規劃 vs 實際」

3. **預算執行**:圓餅圖 · 「人力 / 場地 / 設備 / 其他」實際 vs 預算

**Style**:
- 不用太多色 · 一份報表 ≤ 4 色
- 圖表標題用句子(不是「Chart 1」)· 例「觸及人數較預期超越 40%」
- 軸線標單位(NT$ / 人 / %)

### 章 3 · 見解(2-3 頁 · 客戶該看的)

不要只報「做了什麼」· 要寫「學到什麼 / 推薦什麼」:

```
# 見解 1 · [標題]
觀察:[本案發生了 X 現象]
分析:[原因可能是 Y]
建議:[下次客戶該 Z]

例:
# 見解 1 · IG 互動高於預期 3 倍 · FB 反而下降
觀察:本次活動 IG Reels 互動率 18% · 遠超業界平均 5%。
      但同樣內容在 FB 平均互動 < 1%。
分析:目標族群 25-35 歲已重度使用 IG · FB 對該族群減速。
建議:下次活動可以「IG-first」· 預算 70% 投 IG · FB 改 retargeting 為主。
```

**規則**:
- 2-3 個見解 · 不要 10 個(衝淡)
- 每個見解都要含「客戶可行動」的建議
- 涉及預測時標「預估」(不講死)

### 章 4 · 附錄(可選 · 給工作層看)

放:
- 完整媒體清單(篇名 / 媒體 / 連結 / 日期)
- 完整出席名單(若客戶要)
- 設計稿 / 影片 final 版 連結
- 過程照片(20-50 張 · 含關鍵時刻)
- 收據 / 發票影本(若需報帳)
- 內部交接 SOP(若客戶會自己接著做)

## KPI 計算公式(常用)

### 媒體曝光 AVE(Advertising Value Equivalent)
```
AVE = Σ(該媒體版位廣告價 × 曝光長度比例)
例:聯合報 1/4 版 × 1 篇 = NT$ 80,000
   IG 影音 KOL · 5 萬粉絲 · 1 篇 = NT$ 30,000
總 AVE = 各篇加總
```

### 互動率(Engagement Rate)
```
互動率 = (按讚 + 留言 + 分享 + 收藏) / 觸及人數
業界平均:FB 1-3% · IG 4-8% · LinkedIn 2-4%
```

### ROI(Return on Investment)
```
ROI = (帶來的收入 / 案的總成本) × 100%
例:案 100 萬 · 帶 350 萬合作 · ROI = 350%
注意:「軟性」收入(品牌價值)不算 ROI · 列見解區
```

### 預算執行率
```
執行率 = 實際支出 / 預算 × 100%
標準 95-100% · 過低(< 90%)代表 over-estimate · 也是問題
```

## 紅旗 · 不可寫的內容

絕對不寫:
- 客戶內部人事八卦
- 廠商價格(除非客戶要 · 才匯總不指名)
- 競品具體財務 / 毛利
- 推測性的負面評價(「客戶高層似乎不滿」)
- 沒驗證的數字(「估計」就要標清楚)
